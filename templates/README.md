## Usage Guidelines

1. All templates follow brand guidelines specified in /branding/guidelines/
2. Use approved fonts: Montserrat/Calibri (primary) and Georgia (secondary)
3. Maintain color scheme as defined in /branding/colors/colors.json
4. Follow file naming convention: mport_[category]_[type]_[version]

## Template Access

- Templates are available in multiple formats (DOCX, PPTX, AI, PSD)
- Each template includes usage instructions and examples
- Customizable elements are clearly marked
- Version control is maintained through numbered iterations

## Best Practices

1. Always start with the latest version of templates
2. Maintain consistent spacing and formatting
3. Use approved imagery and graphics
4. Follow content guidelines for tone and messaging
5. Include required legal disclaimers where applicable
